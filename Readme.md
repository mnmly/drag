
# drag

  drag component

## Installation

    $ component install mnmly/drag

## Events
  
  - `dragstart` when drag starts
  - `drag` when dragging
  - `dragend` when drag finishes

## API

  - [Drag()](#drag)
  - [Drag.bind()](#dragbind)
  - [Drag.unbind()](#dragunbind)

## Drag()

  Tuen `el` into draggable element.
  

## Drag.bind()

  Bind event handlers

## Drag.unbind()

  Unbind event handlers
